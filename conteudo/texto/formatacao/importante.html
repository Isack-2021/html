HTML <strong> Tag

Defina texto importante em um documento:

Este é um texto importante - <strong>e este é um texto em importante</strong>.

Definição e uso

A tag <strong> é usada para definir texto com forte importância. O conteúdo dentro é normalmente exibido em negrito.

Pronta: Use a tag <b> para especificar texto em negrito sem qualquer importância extra!
