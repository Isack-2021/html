Formatação de texto html

HTML <b> Tag

Deixe algum texto em negrito (sem marcá-lo como importante):

Este é um texto normal - <b>e este é um texto em negrito</b>.

Este é um texto normal - e este é um texto em negrito.

Definição e uso

A <b> tag especifica texto em negrito sem qualquer importância extra.

Dicas e Notas

Nota: De acordo com a especificação HTML5, a <b> tag deve ser usada como ÚLTIMO recurso quando nenhuma outra tag for mais apropriada. A especificação afirma que os cabeçalhos devem ser indicados com as tags <h1> a <h6> , o texto enfatizado deve ser indicado com a tag <em>, o texto importante deve ser indicado com a tag <strong> e o texto marcado/destacado deve ser denotado com a tag <mark>.

Dica: Você também pode usar o seguinte CSS para definir o texto em negrito: "font-weight: bold;".
