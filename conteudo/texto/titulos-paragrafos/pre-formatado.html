Texto pré-formatado html

O problema do poema

Este poema será exibido em uma única linha:

        <p>
            Minha Bonnie está sobre o oceano.

            Minha Bonnie está sobre o mar.

            Minha Bonnie está sobre o oceano.

            Oh, traga de volta minha Bonnie para mim.
        </p>

Minha Bonnie está sobre o oceano. Minha Bonnie está sobre o mar. Minha Bonnie está sobre o oceano. Oh, traga de volta minha Bonnie para mim.

Solução - O elemento HTML <pre>

O elemento HTML <pre> define o texto pré-formatado.

O texto dentro de um <pre> elemento é exibido em uma fonte de largura fixa (geralmente Courier) e preserva espaços e quebras de linha:

        <pre>
            Minha Bonnie está sobre o oceano.

            Minha Bonnie está sobre o mar.

            Minha Bonnie está sobre o oceano.

            Oh, traga de volta minha Bonnie para mim.
        </pre>
    
        Minha Bonnie está sobre o oceano.

        Minha Bonnie está sobre o mar.

        Minha Bonnie está sobre o oceano.

        Oh, traga de volta minha Bonnie para mim.
