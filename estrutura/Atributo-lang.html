Atributo html lang

O lang atributo especifica a linguagem do conteúdo do elemento.

Exemplos comuns são "en" para inglês, "es" para espanhol, "fr" para francês e assim por diante.

Códigos de linguagem ISO

Você deve sempre incluir o lang atributo dentro da <head> tag, para declarar o idioma da página da Web. Isso é destinado a ajudar os mecanismos de pesquisa e navegadores:

        <html lang="pt">
          ...
        </html>

Códigos de país ISO
