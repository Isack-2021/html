Declaração html

Todos os documentos HTML devem começar com uma declaração <!DOCTYPE>.

A declaração não é uma tag HTML. É uma "informação" para o navegador sobre qual tipo de documento esperar.

Em HTML 5, a declaração é simples:

<!DOCTYPE html>

        <!DOCTYPE html>
        <html>
          <head>
            ...
          </head>
          <body>
            ...
          </body>
        </html>
