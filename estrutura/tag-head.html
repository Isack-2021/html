Marca head

O <head> elemento HTML é um recipiente para os seguintes elementos: <title>, <style>, <meta>, <link>, <script> e <base>.

O <head> elemento é um recipiente para metadados (dados sobre dados) e é colocado entre a tag <head> e a tag <body>.

        <html>
    <head>
            ...
    </head>
        </html>

Metadados HTML são dados sobre o documento HTML. Metadados não são exibidos.

Metadados normalmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras meta informações.
