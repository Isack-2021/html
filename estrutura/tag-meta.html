Tag meta html

Descreva metadados dentro de um documento HTML

A <meta> tag define metadados sobre um documento HTML. Metadados são dados (informações) sobre dados.

<meta> tags sempre entram no elemento <head> e são normalmente usadas para especificar conjunto de caracteres, descrição da página, palavras-chave, autor do documento e configurações de viewport.

Os metadados não serão exibidos na página, mas são parsíveis à máquina.

Metadados são usados por navegadores (como exibir conteúdo ou recarregar página), mecanismos de busca (palavras-chave) e outros serviços web.

Existe um método para permitir que os web designers assumam o controle sobre a configuração (área visível do usuário de uma página da Web), através da tag <meta> "Viewport".

  • <meta charset="UTF-8">
  • <meta name="description" content="breve descrição">
  • <meta name="keywords" conten="palavras-chave>
  • <meta name="author" content="pessoa/empresa">
  • <meta name="viewport" content="width=device-width, initial-scale=1.0">

Especifique a codificação de caracteres para o documento HTML:

<meta charset="UTF-8">

O atributo charset especifica a codificação do caractere para o documento HTML.

A especificação HTML5 incentiva os desenvolvedores web a usar o conjunto de caracteres UTF-8, que cobre quase todos os caracteres e símbolos do mundo!

        <!DOCTYPE html>
        <html>
          <head>
          <meta charset="UTF-8">
          </head>
          <body>
            ...
          </body>
        </html>
