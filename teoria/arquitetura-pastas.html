Estruturando as pastas básica

  • Raiz
    • CSS
    • JS
    • Img
    • index.html

CSS essa pasta vai conter os códigos CSS usados para dar estilo ao seu conteúdo (por exemplo, configurando a cor do texto e do fundo da página).

Js essa pasta vai conter todos os códigos JavaScript usados para adicionar funcionalidades interativas para seu site (ex.: botões que carregam dados quando clicados).

Img essa pasta vai conter todas as imagens que você vai usar no seu site.

índex.html esse arquivo vai geralmente conter o conteúdo da sua página, ou seja, os textos e as imagens que as pessoas veem quando acessam seu site pela primeira vez.
